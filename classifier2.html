How to use it

Uploading a File

Click the "Upload" button to select and upload your file. Wait for the file to upload completely. Your file is successfully if it shown in the box and your file is ready to submit for processing.

Predicting the Closest Result

Once the file is uploaded, click the "Prediction" button to start the classification process. Depending to the strength of your internet connection, the prediction may take some time. Be patient while the application processes the file and generates the prediction.

Displaying the Possible Results

After the prediction process is complete, it will display the closest result or the prediction of corn disease. "Result" box is where the predicted classification is shown. You may now refer to the disease section for the possible actions to treat the disease.
